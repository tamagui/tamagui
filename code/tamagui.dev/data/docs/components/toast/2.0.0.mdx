## Installation

```bash
yarn add @tamagui/toast burnt
```

If targeting native, rebuild your React Native app. Toast relies on [Burnt](https://github.com/nandorojo/burnt) for native functionality.

## Quick Start

The simplest way to use toasts:

```tsx
import { toast, Toaster } from '@tamagui/toast'

function App() {
  return (
    <>
      <Toaster />
      <Button onPress={() => toast('Hello world!')}>Show Toast</Button>
    </>
  )
}
```

## API

### toast()

The imperative toast function:

```tsx
// Basic
toast('Hello world')

// With types
toast.success('Saved!')
toast.error('Failed')
toast.warning('Careful')
toast.info('FYI')
toast.loading('Processing...')

// With options
toast('Message', {
  description: 'More details here',
  duration: 5000,
  action: {
    label: 'Undo',
    onClick: () => undoAction(),
  },
  cancel: {
    label: 'Cancel',
    onClick: () => {},
  },
})

// Dismiss
const id = toast('Hello')
toast.dismiss(id) // dismiss specific
toast.dismiss()   // dismiss all
```

### toast.promise()

For async operations:

```tsx
toast.promise(saveData(), {
  loading: 'Saving...',
  success: 'Saved!',
  error: 'Failed to save',
})

// With data access
toast.promise(fetchUser(), {
  loading: 'Loading user...',
  success: (data) => `Welcome, ${data.name}!`,
  error: (err) => `Error: ${err.message}`,
})
```

### Toaster

The toast container component:

```tsx
<Toaster
  position="bottom-right"  // Position on screen
  expand={false}           // Always show expanded
  visibleToasts={4}        // Max visible toasts
  closeButton={true}       // Show close button
  gap={12}                 // Gap between expanded toasts
  duration={4000}          // Default duration (ms)
/>
```

## Features

### Stacking

Multiple toasts stack visually. Hover over the stack to expand and see all toasts:

```tsx
// Create multiple toasts
toast.success('First')
toast.info('Second')
toast.warning('Third')
```

### Swipe to Dismiss

Toasts can be swiped away. The direction is auto-detected based on position:
- Left/right positioned toasts swipe toward their edge
- Center positioned toasts swipe up/down

### Update Toast

Update an existing toast by ID:

```tsx
const id = toast.loading('Processing...')

// Later...
toast.success('Done!', { id })
```

### Custom Duration

```tsx
// Short duration
toast('Quick!', { duration: 1000 })

// Persistent (no auto-dismiss)
toast.loading('Processing...', { duration: Infinity })
```

## Legacy API (v1)

The previous declarative API is still available for backwards compatibility:

```tsx
import { Toast, ToastProvider, ToastViewport, useToastController } from '@tamagui/toast/v1'
```

See the [v1 documentation](/docs/components/toast/1.83.0) for the legacy API.
